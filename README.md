# tp-ciencia-datos-2026

Trabajo Práctico de Ciencia de Datos 2026.

## Estructura del proyecto

```
tp-ciencia-datos-2026/
├── data/
│   ├── raw/              # Datos originales (sin procesar)
│   └── processed/        # Datos limpios o transformados
├── notebooks/
│   ├── entrega_1.ipynb
│   ├── entrega_2.ipynb
│   └── entrega_3.ipynb
├── src/                  # Scripts reutilizables
│   ├── data_cleaning.py
│   ├── utils.py
│   └── visualization.py
├── reports/
│   ├── figures/          # Gráficos exportados
│   └── videos/           # Links o referencias a videos
├── README.md
├── requirements.txt
└── .gitignore
```

## Instalación

```bash
pip install -r requirements.txt
```
