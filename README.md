## 📌 Descripción

Proyecto integrador de la materia **Ciencia de Datos**, donde se desarrolla un flujo completo:

- 🧠 Formulación del problema  
- 📥 Ingesta y comprensión de datos  
- 📊 Análisis exploratorio (EDA)  
- 🤖 Evaluación de modelado  

El proyecto está dividido en **3 entregas acumulativas**.

## 📂 Dataset

### ✅ Requisitos

* ≥ 500 filas
* ≥ 8 columnas
* Variables numéricas y categóricas
* Datos con imperfecciones reales
* Fuente pública

### 🌐 Posibles fuentes

* Kaggle
* UCI Machine Learning Repository
* Datos abiertos Argentina
* Banco Mundial

---

## 📦 Entregas

---

### 📘 Entrega 1 — Problema y datos

**Objetivo:** Definir el problema y comprender el dataset

**Incluye:**

* Contexto del problema
* Preguntas analíticas
* Carga de datos (Pandas)
* Análisis inicial de calidad
* Justificación del dataset

---

### 📊 Entrega 2 — EDA

**Objetivo:** Analizar los datos en profundidad

**Incluye:**

* Limpieza de datos
* Análisis descriptivo
* Correlaciones
* ≥ 5 visualizaciones
* Síntesis de hallazgos

---

### 🤖 Entrega 3 — Modelado

**Objetivo:** Evaluar si es viable modelar

#### 🟢 Modelar

* Tipo de modelo
* Variable objetivo
* Justificación

#### 🔴 No modelar

* Limitaciones
* Alternativas analíticas

---

## 📊 Evaluación

* 📌 Peso: **40% de la nota final**
* 📈 Promedio de las 3 entregas
* ✅ Promoción:

  * Cada entrega ≥ **8**
